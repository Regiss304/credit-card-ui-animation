# Credit Card Flip UI

## 📌 Objective
This project demonstrates a 3D flipping card animation using HTML and CSS.  
The goal is to practice advanced CSS concepts such as transforms, perspective, and visual effects.

---

## 🚀 Features
- Front and back card design
- 3D flip animation on hover
- Use of CSS perspective and transform
- Glassmorphism-style card design
- Structured layout using Flexbox

---

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox, transforms, perspective, animations)
- Google Fonts (Poppins)

---

## 📚 What I Learned
- How to create 3D animations using CSS
- Working with `transform`, `rotateY`, and `perspective`
- Using `backface-visibility` to control element rendering
- Structuring layered UI components
- Applying modern visual styles like glassmorphism

---

## ⚠️ Limitations
- No responsiveness (fixed width and height)
- No interactivity beyond hover
- Static content (no dynamic data)
- Not usable as a real product in current state
- No accessibility considerations

---

## 🔧 Future Improvements
- Add responsiveness with media queries
- Implement dynamic data (e.g., user input or API)
- Add JavaScript interaction (click instead of hover)
- Improve accessibility (semantic HTML, ARIA labels)
- Convert into a reusable component (React)
